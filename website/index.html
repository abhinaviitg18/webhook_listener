Quick Start

1. Create a listener to get your unique ingress URL.
2. Point your app's webhooks at that URL.
3. Define Skills to classify and filter incoming data.
4. Configure Integrations (like OpenClaw or Slack) to receive high-signal events.

POST /ingest/{account_slug}/{provider}/{secret}
Content-Type: application/json

{ "event": "user.signup", "email": "bob@example.com" }
